# ⚔️ 수학 대전 (Math Battle)

초등학교 3~4학년 대상 1:1 멀티플레이어 연산 배틀 게임

## 🎮 게임 소개

**수학 대전**은 두 명의 플레이어가 각자 자신의 기기(스마트폰, 태블릿, PC)에서 접속하여 수학 연산 문제를 풀며 대결하는 실시간 멀티플레이어 게임입니다.

### 주요 기능

- **🔑 로그인 시스템**: 아이디/비밀번호 기반 계정 관리, 포인트 누적, TOP 10 랭킹
- **📱 1인 1기기**: PeerJS(WebRTC) P2P 통신으로 별도 서버 없이 실시간 대전
- **🎭 40종 캐릭터**: 다양한 동물/판타지 캐릭터 선택
- **🏠 방 시스템**: 4자리 코드로 쉽게 방 만들기/입장, 5초 카운트다운
- **🏳️ 패배 인정**: 중도 포기 시 포인트 50% 차감 경고 시스템

## 🎮 게임 모드

| 모드 | 설명 |
|------|------|
| ⚡ **스피드 모드** | 30초 안에 더 많이 맞히면 승리! 빠른 판단력이 핵심 |
| 🛡️ **서바이벌 모드** | HP를 깎아 상대를 쓰러뜨려라! 아이템을 전략적으로 활용 |
| 🐉 **보스 대전 모드** | 2명이 협력하여 보스를 물리쳐라! 함께 문제를 풀어 보스 HP를 깎자 |

## 🏆 포인트 시스템

- 승리 기본: `50 × (난이도+1)`
- 남은 HP 보너스: `HP × 20`
- 퍼펙트 보너스: `+100` (HP 손실 없이 승리)
- 패배 인정 패널티: 보유 포인트의 **50% 차감**

## 🚀 사용 방법

### 방법 1: 직접 열기
`index.html` 파일을 웹 브라우저에서 직접 열기

### 방법 2: GitHub Pages
1. 이 저장소를 GitHub에 업로드
2. Settings → Pages → Source: `main` 브랜치 선택
3. 생성된 URL을 학생들에게 공유

### 교실에서 사용하기
1. 같은 Wi-Fi 네트워크에 연결
2. 학생 A: 회원가입 → 방 만들기 → 방 코드 공유
3. 학생 B: 회원가입 → 방 코드 입력 → 입장
4. 양쪽 캐릭터 선택 → 준비 완료 → 5초 후 게임 시작!

## 🛠️ 기술 스택

- **프론트엔드**: Vanilla JavaScript, CSS3
- **실시간 통신**: PeerJS (WebRTC)
- **데이터 저장**: localStorage (계정, 포인트)
- **오디오**: Web Audio API

## 📐 연산 범위 (3~4학년)

- ➕ 덧셈: 두 자릿수 + 두 자릿수
- ➖ 뺄셈: 두 자릿수 - 두 자릿수
- ✖️ 곱셈: 구구단 범위 (난이도에 따라 확장)
- ➗ 나눗셈: 나누어떨어지는 수
